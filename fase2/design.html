Sintaxis Css Interacción Humano Ordenador

Las Hojas de estilo o CSS, estas permiten a HTML ampliar el lenguaje, por medio de su presentacion visual, esto es posible ya que e formato CSS permite la vinculacion con los Css, esto hace que se divida en dos partes el desarrollo web, la parte HTML y el diseño del mismo.

¿Que aporta separar la estructura del diseño? se usaria menos codigo y seria mas transparente, ya que en lugar de tener un codigo enerme con etiquedas, bastante complejas segun el nivel del programador, tendria un codigo legible ya que se encontraria con etiquedas HTML por un lado y en el otro la forma de desarrollo de Css.

El Css u hojas de estilo consta de varias partes:
1 selector posteriormente abrimos corchetes.
2 un bloque de declaracinión. El bloque de declaracion tiene varias partes que son:
    2.0 la primera es la propiedad, terminandola con dos puntos.
    2.1 es el valor referente a esa propiedad, separados por coma y terminandola con punto y coma ";"

Selectores Css Interació Humano Ordenador

Los selectores definen sobre que elementos o etiquetas se aplicara el estilo, es decir la semantica,normas ,protocolos:
1: selector de tipo --> este selecciona todas las etiquetas que sean de determinado tipo ejemplo
table se aplica el cambio a cualquier etiqueta que tenga la palabra reservada table.
2: selector de clase: --> selecciona las etiquetas que tienen el atributo "class" usando punto mas el nombre de a clase.
input type="text" method="post" class="nombre" --> .nombre{}
3: selector de Id: selecciona un elemento con el atributo id usando un # numeroal como referencia. input type="text" method="post" id="nombre" --> #nombre{}
